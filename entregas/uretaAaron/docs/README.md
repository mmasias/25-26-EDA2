# Tarea de Recursividad

Esta tarea reúne una serie de ejercicios prácticos para comprender y aplicar **recursividad** en diferentes problemas. Cada ejercicio incluye:

- **Descripción del problema**: lo que se busca calcular o transformar.
- **Análisis recursivo**: explicación paso a paso de cómo se resuelve utilizando recursión.
- **Pseudocódigo**: algoritmo general que implementa la solución recursiva.

## Ejercicios incluidos

1. **Contar cifras de un número** – Determinar cuántos dígitos tiene un número entero.
2. **Convertir a binario** – Representar un número en sistema binario.
3. **Duplicar elementos de una lista** – Generar una lista con cada elemento repetido dos veces.
4. **Elevar al cuadrado** – Calcular el cuadrado de un número sin usar multiplicación directa.
5. **Eliminar todas las apariciones de una letra** – Quitar una letra específica de una cadena.
6. **Validar palíndromo** – Comprobar si una palabra se lee igual al derecho y al revés.
7. **Producto de pares hasta n** – Multiplicar todos los números pares desde 2 hasta *n*.
8. **Reemplazar una letra por otra** – Sustituir todas las apariciones de una letra por otra en una cadena.
9. **Sumar elementos en posiciones pares** – Sumar los elementos de una lista que están en índices pares.

Cada ejercicio sirve para practicar el pensamiento recursivo y entender cómo se puede dividir un problema en subproblemas más pequeños que se resuelven de forma similar.
