# Práctica – Recursividad (Sección 2Think)

## Descripción

En esta entrega se implementan distintos algoritmos recursivos en Java con el objetivo de practicar:

- Diseño recursivo
- Caso base
- Reducción del problema
- Procesamiento de números, cadenas y listas

---

## Ejercicios implementados

1. Producto de pares hasta n  
2. Contar cifras de un número  
3. Elevar n al cuadrado  
4. Eliminar apariciones de una letra  
5. Reemplazar una letra  
6. Validar palíndromo  
7. Duplicar elementos de una lista  
8. Sumar elementos en posiciones pares  
9. Conversión decimal a binario
